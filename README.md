# MyChat
Computer Network Course Project

Compile:
1.首先先在根目录下创建bin资料夹
2.执行build.sh
  `sh build.sh`
Run:
Server:`sh runServer.sh`  
Client:`sh runClient.sh`
   
Environment Requiredment:  
Windows/Mac OS X/Linux  
Java JDK 1.8.0 or later

功能简介:
在线群聊
P2P聊天
群发文件
P2P发送文件

Notice:
直接从github下载下来的原码可能在windows下执行会有编码问题
